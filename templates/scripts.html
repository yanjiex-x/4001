Name User Guide
  • microsoftapps.ps1

    • or

  • microsoftapps.bat
 Script to update all Microsoft Applications.

Note: Not all applications can be updated.
The winget tool is only supported on Windows 10
1709 or later.

For first-time winget users, enter the following command:

winget upgrade


Windows Powershell
To execute the script, enter:

./microsoftapps.ps1

or

./microsoftapps.bat


Windows Command Prompt
To execute the script, enter:

microsoftapps.bat
